## Списки

Чтобы добавить ненумерованные списки нужно выделить с одной столроны звёздочкой (*). Например вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки нужно в начале написать цифру (1.). Например вот так:
1. элемент 1
2. элемент 2
3. элемент 3
